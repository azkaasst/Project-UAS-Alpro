## 1. Nama Program

Aplikasi Kasir Coffee Shop

## 2. Deskripsi Program

Program ini dibuat menggunakan bahasa pemrograman java dengan tampilan GUI hasil import dari library java swing. Program ini dibuat karena saya memiliki cita cita untuk membuat suatu perusahaan yang bergerak di bidang kopi yaitu coffeeshop yang mana program ini akan saya implementasikan terhadap perusahaan saya pada sistem kasirnya, program ini masih belum sempurna dikarenakan masih ada beberapa kekurangan pada tampilan nya, tetapi program ini dapat dikembangkan lagi menjadi program yang dapat mudah dipakai oleh semua orang dan mudah dimengerti. Tujuan dibuat nya program aplikasi kasir ini untuk membuat para customer coffeshop merasakan user experience pada coffeeshop saya dan dapat merasakan fitur yang saya buat pada aplikasi kasir ini.
